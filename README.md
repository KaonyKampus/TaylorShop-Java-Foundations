# TaylorShop - Inventory Management System 🛒

This project is a console-based application developed in **Java** as part of my preparation for the **Java Foundations** certification. It serves as a practical exercise to master core programming logic and data handling.

## 🚀 Features (Version 1.0)
* **Product Registration:** Allows input for name, stock, and price for up to 3 products.
* **Dynamic Discount Calculation:** Automatically applies a **15% discount** if the product price is $100,000 or higher.
* **Inventory Consultation:** Displays detailed information for each registered product, including calculated discount values.
* **Data Deletion:** Provides an option to clear specific product data.
* **Interactive CLI:** A user-friendly menu built with nested `switch` statements and `while` loops.

## 🛠️ Java Concepts Applied
During this challenge, I focused on mastering the following topics from the certification exam:
* **Control Structures:** Implementation of nested `switch` cases and boolean logic for flow control.
* **Data Types:** Management of primitive types (`int`, `double`) and `String` objects.
* **Scanner Class:** Handling console input and managing the input buffer.
* **Constants:** Utilizing the `final` keyword for fixed values like tax and discount rates.
